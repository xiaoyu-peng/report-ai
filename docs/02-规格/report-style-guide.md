# 报告风格指南（5 种样例模板总结）

> 来源：`docs/以稿写稿-报告样例/` 文件夹下 5 份 PDF 样例 + `报告样例说明.pdf`。
> 目的：为"风格学习"提示词和"仿写"报告模板提供参考蓝本。

## 典型报告类型与应用场景

| 报告类型 | 业务场景 | 典型客户 | 使用频率 |
| --- | --- | --- | --- |
| 政策影响分析报告 | 解读政策变化对行业/企业的影响 | 政府机构、咨询公司 | 政策发布时 |
| 行业分析报告 | 分析行业趋势、竞争格局、技术演进 | 投资机构、战略部门 | 季度 / 年度 |
| 传播分析报告 | 评估品牌传播效果、舆情监测 | 公关公司、市场部 | 月度 / 季度 |
| 科技情报专题报告 | 跟踪技术发展、专利分析、竞品研究 | 研发部门、情报部门 | 月度 / 专项 |
| 专题日报 / 周报 | 定期追踪特定主题的动态变化 | 高管、业务团队 | 每日 / 每周 |

## 模板结构速查

### 1. 政策影响分析报告（样例：房地产政策多维调整）
- **一、事件概述**（政策发布的时间、主体、文件号、核心意图，一段 300-500 字的概括）；
- **二、热点梳理**（列多个子政策，用「（一）」「（二）」结构，每条给"事件概述 + 政策要点 + 适用人群/地域"）；
- **三、影响分析**（分行业 / 分主体拆解）；
- **四、结论与建议**。
- 语气：客观、克制、引用官方文号。页脚有"内部参考，请勿传播"。

### 2. 行业分析报告（样例：AI 赋能文旅产业）
- **开场总述**（一段宏观背景，3-5 行交代产业现状和技术变革）；
- **一、"AI+文旅"领域的新趋势与挑战**
  - （一）文旅企业积极拥抱，重点探源城市竞相接入 DeepSeek；
  - （二）AI 技术未来将与科技 / 文化 / 教育等领域深度融合；
  - （三）数字文旅影响消费升级加速演进，AI 做攻略成为年轻群体旅游新宠儿；
  - （四）技术渗透与场景痛点并存深度突出……
- **二、竞争格局 / 典型案例**；
- **三、发展建议 / 研判**。
- 语气：研判性、多角度、穿插数据与案例，引用"中国青年网"、"智能旅游平台"等出处。

### 3. 传播分析报告
- **一、事件回顾**（时间线）；
- **二、传播声量 / 渠道分布**（典型图表：声量折线、渠道饼图、情感分布）；
- **三、关键节点与热文**；
- **四、媒体声量 / 网民观点**；
- **五、舆情演化与建议**。
- 强数据驱动，图表密集。MCP 的 `sass-mcp` 完整覆盖所需指标。

### 4. 科技情报专题报告
- 以"技术大纲"组织：技术背景 / 主要进展 / 代表公司 / 专利布局 / 竞品研究 / 未来判断。
- 通常含"大纲模板"页——建议直接作为"风格学习"的骨架。

### 5. 专题日报 / 周报
- 结构最轻：**今日要闻 → 三条重点 → 数据快讯 → 动向观察**。
- 短平快，500-1500 字。

## 共性写作风格

- **标题**：宽泛主题 + 「：」+ 具体切入，或"关于 XX 的分析与研判"；
- **章节**：顶层用"一、二、三"，次级用"（一）（二）（三）"；
- **语气**：第三人称、书面中文，避免"我们认为"，改用"业内认为 / 数据显示"；
- **引用**：行内用【】或"（来源：XXX 报告）"标注来源；
- **数据**：必附百分比、绝对数或时间戳，否则被判为"未落地";
- **结尾**：必须有结论 + 建议，不能戛然而止。

## 风格学习 Prompt 模板（给 Claude API）

```
你是一位资深报告分析师。下面给你一份参考报告全文，请抽取它的写作风格与结构，输出 JSON：
{
  "report_type": "政策影响 | 行业分析 | 传播分析 | 科技情报 | 专题日报",
  "section_hierarchy": ["一、XXX", ["（一）...", "（二）..."], ...],
  "tone": "正式 / 研判 / 速报",
  "citation_pattern": "脚注编号 | 行内括号 | 来源标注",
  "avg_paragraph_length_zh": 250,
  "data_density": "高 / 中 / 低",
  "signature_phrases": ["业内认为", "数据显示", ...]
}
```

后续"仿写"阶段把这个 JSON 注入系统提示，配合检索到的知识库 chunk 一起送给模型。

## 对 UI 的要求

报告预览区必须支持：
- 章节折叠（一级 / 二级）；
- 行内引用高亮（hover 展示知识库来源）；
- 修订痕迹对比（改写模式需要）。
